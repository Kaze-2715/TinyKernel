# TinyKernel
既然在学操作系统，为什么不试着写一个小小的操作系统内核呢？  
- 真内核
- 使用 RISC-V 指令集  
---
# 开发计划
课程1：环境搭建与裸机启动
- 安装RISC-V交叉编译工具链
- 配置QEMU riscv64-virt运行环境
- 编写最小汇编入口，理解链接器脚本
- 目标：QEMU启动后能看到串口输出字符

课程2：C语言接管与格式化输出
- 从汇编跳转到C代码
- 实现简易UART驱动（轮询方式）
- 实现printk函数支持%c %s %d %x
- 目标：能用C语言打印调试信息

课程3：RISC-V特权架构基础
- 理解M-mode与S-mode的区别
- 设置S-mode的stvec陷阱向量基址
- 实现简单的异常捕获与死循环报告
- 目标：非法指令能触发自定义异常处理

课程4：中断控制器PLIC
- 初始化PLIC并启用UART外部中断
- 理解中断优先级与阈值
- 实现中断使能与屏蔽
- 目标：串口输入能触发中断处理函数

课程5：时钟中断与抢占基础
- 配置RISC-V定时器（mtime/mtimecmp）
- 实现时钟中断处理程序
- 理解中断嵌套与上下文保存概念
- 目标：每秒固定次数的时钟中断

课程6：上下文切换汇编
- 设计任务上下文结构（保存ra/sp/s0-s11等）
- 编写保存上下文与恢复上下文的汇编代码
- 理解调用约定与寄存器使用规则
- 目标：能手动触发两个任务的切换

课程7：简单调度器
- 设计进程控制块（PID、状态、栈、上下文指针）
- 实现轮转调度算法
- 创建两个线程循环打印不同字符
- 目标：时钟中断自动切换两个任务

课程8：物理内存管理
- 获取QEMU传递的内存信息（device tree或硬编码）
- 实现位图或伙伴算法的物理页分配器
- 实现页框的申请与释放
- 目标：能动态分配4KB页框并验证不重叠

课程9：页表与虚拟内存
- 理解RISC-V Sv39页表三级结构
- 实现页表遍历与映射建立
- 开启分页（satp寄存器设置）
- 目标：内核在开启分页后仍能正常运行

课程10：用户态与系统调用
- 设计系统调用接口（ecall指令）
- 实现用户态栈与内核栈分离
- 实现简单的syscall_handler分发
- 目标：用户程序能调用sys_putchar输出字符

课程11：进程创建与销毁
- 实现fork的简化版（复制地址空间与上下文）
- 实现exec的简化版（替换地址空间）
- 实现exit与资源回收
- 目标：shell能执行新程序

课程12：同步原语
- 实现自旋锁（基于原子指令）
- 实现信号量或互斥锁
- 解决简单的生产者消费者问题
- 目标：多任务安全共享缓冲区
